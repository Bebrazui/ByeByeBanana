Это форк оригинального ByeByeDPI, созданный для тех, кому лень каждый раз нажимать кнопку "Старт". Больше автоматизации, больше бананов, меньше цензуры.
### 🚀 Главные фичи:
Auto-App Launch: Приложение само просыпается, когда ты заходишь в YouTube, Discord или любое другое приложение из твоего списка.
Smart Engine Control: Выбор между Accessibility Service и Usage Stats для отслеживания активности.
Adaptive Connection: Авто-отключение, если провайдер слишком сильно портит пинг.
### 🛠 Как собрать
Склонируй репозиторий.
Установи Android NDK и CMake.
Нажми Build APK в Android Studio и молись, чтобы JNI не выдал ошибку.
### ⚠️ Disclaimer
Проект создан в образовательных целях и ради рофла. Название пакета com.byebyebanana фиксированное, так что если решите ставить поверх оригинала — не выйдет, это отдельная сущность.
